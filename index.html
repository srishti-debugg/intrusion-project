Intrusion Detection Website

Learn and prototype IDS workflows using classic ML (PCA + Random Forest) and deep learning methods with Python and Java.

Architecture Overview

  1. Data Ingestion: Capture network flow records.
  2. Preprocessing: Normalize and encode features.
  3. Feature Reduction: PCA for lower-dimensional features.
  4. Classical Model: Random Forest for interpretable baseline.
  5. Deep Learning Model: Autoencoder / LSTM for complex attack patterns.
  6. Serving Layer: Python model API consumed by Java app.

Python (Model Training)

  • Use Scikit-learn for PCA + Random Forest.
  • Use TensorFlow or PyTorch for deep learning.
  • Expose predictions with FastAPI/Flask.

Java (Application Layer)

  • Build dashboard/API gateway with Spring Boot.
  • Integrate alerts, user auth, and reporting.
  • Call Python inference endpoints via REST.

Python Example (Training)

from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
from sklearn.ensemble import RandomForestClassifier

rf_pipeline = Pipeline([
    ("scaler", StandardScaler()),
    ("pca", PCA(n_components=12)),
    ("rf", RandomForestClassifier(n_estimators=300))
])

Deep Learning Example (Python)

import tensorflow as tf

model = tf.keras.Sequential([
    tf.keras.layers.Input(shape=(30,)),
    tf.keras.layers.Dense(64, activation="relu"),
    tf.keras.layers.Dense(32, activation="relu"),
    tf.keras.layers.Dense(1, activation="sigmoid")
])

model.compile(optimizer="adam", loss="binary_crossentropy", metrics=["AUC"])
